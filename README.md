
# 🎲 말랑말랑 보드게임 (Interactive Educational Board Game)

## 📝 프로젝트 개요
**말랑말랑 보드게임**은 초등학교 교실의 전자칠판 및 태블릿 환경에 최적화된 **단일 HTML 기반의 인터랙티브 교육용 웹 보드게임**입니다. 
아이들의 흥미를 끄는 '클레이모피즘(Claymorphism)' 디자인을 적용하였으며, 교사가 직접 학습 문제를 출제하고 모둠별 시합을 진행할 수 있도록 설계되었습니다. 별도의 서버(백엔드)나 복잡한 설치 없이 브라우저만 있으면 즉시 실행 가능합니다.

## ✨ 핵심 UI/UX 디자인
* **클레이모피즘(Claymorphism) 100% 적용:** 폭신하고 부드러운 찰흙 질감의 3D 버튼과 컴포넌트로 아이들에게 친근한 시각적 경험 제공.
* **완벽한 반응형 & 터치 최적화:** TV, 모니터, 태블릿 등 다양한 해상도에 맞춰 말판과 여백이 동적으로 계산되며 꽉 차게 렌더링 됨. 터치 딜레이 방지(`touch-action: manipulation`) 적용.
* **시인성 극대화:** 팝업 창 등장 시 배경 블러(Blur) 및 어둡게 처리, 캐릭터 토큰 테두리 및 그림자 강조로 가독성 및 집중도 향상.

## 🎮 주요 게임 기능
### 1. 다중 플레이 및 캐릭터 시스템
* **1인 ~ 6인 로컬 멀티플레이:** 개인전부터 모둠전(최대 6팀)까지 설정 가능.
* **동물 캐릭터 토큰:** 6가지 귀여운 동물 이모지(🐰, 🐱, 🐼, 🐸, 🐷, 🦄)를 각 플레이어에게 할당.
* **스마트 겹침 방지 알고리즘:** 여러 플레이어의 말이 같은 칸에 도착해도 겹치지 않고 자동으로 옹기종기 흩어져서 예쁘게 클러스터링 됨.
* **끝장 승부 로직:** 1등이 도착해도 게임이 끝나지 않고, 마지막 1명이 남을 때까지 나머지 플레이어들끼리 턴이 지속됨.

### 2. 실감 나는 애니메이션 & 사운드
* **중앙 3D 주사위:** 화면 중앙의 3D 주사위를 터치하면, CSS 3D Transform을 활용해 실감 나게 구르며 랜덤 숫자를 반환.
* **토큰 점프 애니메이션:** 이동 시 해당 플레이어의 말만 부드럽게 통통 튀며(`bounce`) 칸을 이동함.
* **내장 사운드 엔진 (Web Audio API):** 외부 오디오 파일 없이 웹 브라우저 자체 합성음으로 주사위 구르는 소리, 점프 소리, 정답(띠링), 오답(땡) 사운드 구현 (Mute 옵션 제공).

## 📚 교사 맞춤형 교육(퀴즈) 시스템
* **시크릿 관리자 대시보드:** 우측 상단의 톱니바퀴(⚙️)를 눌러 언제든 문제를 출제하고 실시간으로 말판에 반영 가능.
* **칸 번호 매핑:** 1번 칸, 5번 칸 등 특정 칸에 원하는 문제를 지정하여 배치 가능. 지정하지 않은 칸은 '랜덤 수학 연산 문제(덧셈)'가 자동 출제됨.
* **확산적 사고 (오픈엔디드) 모드:** 교사가 정답 칸을 비워두면, 입력창 대신 [🗣️ 발표 완료] 버튼만 나타나 학생이 자유롭게 말이나 행동으로 정답을 표현하고 넘어갈 수 있음.
* **전자칠판 전용 숫자 키패드:** 화면 내에 큼직한 가상 숫자 키패드를 자체 내장하여 터치스크린에서의 입력 편의성 극대화.
* **타이머 기능:** 15초, 30초, 무제한 등 문제 풀이 제한 시간 설정 가능 (타임바 UI 제공).

## 🗺️ 2가지 맵 테마 지원
### 1️⃣ 자연의 섬 (지그재그 모드)
* **특징:** 섬들이 다리로 연결되어 뱀처럼 굽이치는 캐주얼한 디자인.
* **옵션:** 15칸, 20칸, 30칸 중 길이를 선택하여 동적으로 맵 생성.

### 2️⃣ 블루마블 (프리미엄 28칸 모드)
* **특징:** 직사각형 테두리를 따라 도는 보드게임의 정석. 중앙에 고급스러운 투명 로고 배치.
* **말판 텍스트 인쇄 & 숨기기:** 말판 안에 해당 칸의 문제가 인쇄되어 보임. 긴 문제도 스마트하게 자동 줄바꿈 및 말줄임표 처리. 교사 설정에 따라 모든 문제를 '❓'로 숨겨 긴장감을 유발할 수 있음.
* **특수 칸 (Special Spaces):**
  * `[0번 칸] 🏁 출발`: 한 바퀴를 완주하여 통과하거나 도착하면 승리.
  * `[9번 칸] 🏝️ 무인도`: 도착 시 아무 조치 없이 '다음 1턴 강제 휴식' 페널티 부여.
  * `[14번 칸] 🎤 장기자랑`: 정답이 없는 확산적 사고 칸으로 강제 배정.
  * `[23번 칸] 🌪️ 앗! 처음으로`: 도착 시 0번 칸으로 강제 소환.

## 🛠️ 기술 스택 (Tech Stack)
* **Frontend:** HTML5, CSS3, Vanilla JavaScript
* **특이사항:** Zero Dependencies (외부 라이브러리, 프레임워크, 이미지, 사운드 파일 전혀 없이 순수 코드 1개의 HTML 파일로 모든 로직 구현)
